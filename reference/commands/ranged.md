# /ranged

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/ranged [ <spawnID> ]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Performs a ranged attack.
<!--cmd-desc-end-->

## Notes

* No parameter executes a ranged attack on your current target
* If _spawnID_ is supplied this will attempt to execute a ranged attack on that spawn if it exists.
