# /crash

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/crash [force]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Create a synthetic crash for debugging purposes.
<!--cmd-desc-end-->
## Examples
- `/crash` - Creates a synthetic crash dump
- `/crash force` - Forces an immediate access violation crash

Note: Requires crash submissions to be enabled
