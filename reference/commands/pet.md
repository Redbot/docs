# /pet

## Syntax

```eqcommand
/pet <action> <spawnid>
```

## Description

MacroQuest extends EverQuest's `/pet` command to allow attacking by spawn #.

## Options

- **attack** _#_ - Can specify the spawn ID number to attack
- **qattack** _#_ - Can specify the spawn ID number to queue an attack

## Examples

```text
/pet attack 1234
/pet qattack 1234
```
