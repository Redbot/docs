# /identify

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/identify
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Displays further information about the item on the cursor, similar to the spell _Identify_ and item lore displayed by MQ in the item stats UI window.
<!--cmd-desc-end-->
