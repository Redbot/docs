# /items

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/items [<filter>]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
* Lists all ground spawns and environmental containers in the zone that match _filter_
* All items will be listed if no filter is supplied
<!--cmd-desc-end-->
