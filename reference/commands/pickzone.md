# /pickzone

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/pickzone <#>
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Extends EverQuest's `/pickzone` command to allow picking a zone by number. 0 is the main instance.
<!--cmd-desc-end-->
## Examples

- **/pickzone 0** - Main Instance
- **/pickzone 1** - Picks zone 1
