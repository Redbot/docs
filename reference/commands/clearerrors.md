# /clearerrors

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/clearerrors
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Clears each of the last errors in the [_macroquest_](../../reference/data-types/datatype-macroquest.md) type:

* **${MacroQuest.Error}**
* **${MacroQuest.MQ2DataError}**
* **${MacroQuest.SyntaxError}**
<!--cmd-desc-end-->
