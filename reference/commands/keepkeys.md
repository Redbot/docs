# /keepkeys

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/keepkeys [off|on]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Keeps keys that were pressed with [/keypress](keypress.md), in their current state when a macro ends. /keepkeys with no arguments displays the current on/off state.
<!--cmd-desc-end-->
