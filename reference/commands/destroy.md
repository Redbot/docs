# /destroy

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/destroy
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Destroys whatever you have on your cursor with no confirmation, **even if you have "Destroy Confirmation" enabled in your EQ options**. **Use with care.**
<!--cmd-desc-end-->
## Examples

```text
/if ( ${Cursor.Name.Find[rusty]} ) /destroy
```
