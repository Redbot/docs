# /buyitem

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/buyitem [<quantity>]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Purchases the currently selected item from a merchant window. You must have an item selected in the merchant window for this command to work.
<!--cmd-desc-end-->
## Examples

- `/buyitem 20` - Buys 20 of the currently selected item
