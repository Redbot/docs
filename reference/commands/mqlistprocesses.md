# /mqlistprocesses

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/mqlistprocesses [<name>]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
List relevant processes to help debug stuck and/or broken dependencies/plugins. `<name>` is a filter.
