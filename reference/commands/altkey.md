# /altkey

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/altkey <command>
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Execute _command_ while telling the window manager that the alt key is pressed.
<!--cmd-desc-end-->

## Note
Can also be used together with [/ctrlkey](ctrlkey.md) , [/shiftkey](shiftkey.md) , or both as in:

```text
 /ctrlkey /altkey /shiftkey command
```
