# /foreground

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/foreground
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Brings the EverQuest game window that entered the command to the front of your desktop. Useful when switching between multiple EQ windows or other applications.
<!--cmd-desc-end-->
