# /getwintitle

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/getwintitle
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Retrieves and displays the current window title of the EverQuest client window. This is useful for debugging title changes made with [/setwintitle](setwintitle.md).
<!--cmd-desc-end-->
