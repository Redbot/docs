# /location

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/location
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Extends the EverQuest Y, X, Z coordinates to add the direction you're facing from a 16-wind compass.
<!--cmd-desc-end-->
## Example Output

```text
Your Location is 100, 200, 10, and are heading west by southwest.
```
