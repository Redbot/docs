# /memspell

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/memspell <gem> "<spellname>"
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Attempts to memorize "_spellname_" into gem _gem_. Any _spellname_ with more than one word must be surrounded by quotes.
<!--cmd-desc-end-->
## Example

`/memspell 1 "Shallow Breath"`
