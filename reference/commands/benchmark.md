# /benchmark

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/benchmark [<command>]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Produces a list of all active benchmarks, or times the execution of a command.
<!--cmd-desc-end-->
## Examples
| Command | Description |
| :--- | :--- |
| **/benchmark /who all** | Times the execution of the `/who all` command |
