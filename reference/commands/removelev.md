# /removelev

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/removelev
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Will attempt to remove any levitation buffs on you that are found in the buff or shortbuff window.
<!--cmd-desc-end-->
## Example

```text
 /removelev
```
