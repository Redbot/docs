# /itemtarget

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/itemtarget "<itemname>"
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
"Targets" a ground spawn or environmental container. The item targeted will not show up in the target window.
<!--cmd-desc-end-->
