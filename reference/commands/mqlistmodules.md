# /mqlistmodules

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/mqlistmodules [<name>]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
List loaded modules in the MQ directory to help with debugging stuck and/or broken dependencies/plugins. `<name>` is a filter.
<!--cmd-desc-end-->
