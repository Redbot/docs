# /itemslots

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/itemslots
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Lists available item slots, including equipped, inventory, bank, shared, current container, and so on. Output is in the following format:
```text
[index]: [container location] [slot numbers] [item name]
```
<!--cmd-desc-end-->
