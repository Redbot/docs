# `PointMerchant`

<!--tlo-desc-start-->
Access to point merchants (such as those found in LDoN) when a window is open.
<!--tlo-desc-end-->

## Forms
<!--tlo-forms-start-->
### {{ renderMember(type='pointmerchant', name='PointMerchant') }}

:   Returns TRUE if point merchant window is open
