## Usage

```
/echo ${Group.Leader.ID}
```

Echo Groupleader ID, if any.

```
/echo ${Group.Member[0]}
```

Echos your own name

```
/echo ${Group.Member[1]}
```

Echos the next person on the list, after yourself.
