# `String`

<!--tlo-desc-start-->
Access the string datatype
<!--tlo-desc-end-->

## Forms
<!--tlo-forms-start-->
### {{ renderMember(type='string', name='String', params='text') }}

:   Access text as a string

<!--tlo-forms-end-->

## Examples

!!! example
    `${String[foo].Right[2]}`
    : Returns the right two characters of "foo", which is "oo".
