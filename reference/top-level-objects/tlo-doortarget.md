# `DoorTarget`

<!--tlo-desc-start-->
Object used to return information on your doortarget.
<!--tlo-desc-end-->
## Forms
<!--tlo-forms-start-->
### {{ renderMember(type='switch', name='DoorTarget')}}

:   Returns the currently targeted door/switch
<!--tlo-forms-end-->

## Usage

```text
/echo ${DoorTarget.ID}
```

Returns the ID of your door target.
