# `GameTime`

<!--tlo-desc-start-->
A time object indicating EQ Game Time.
<!--tlo-desc-end-->
## Forms
<!--tlo-forms-start-->
### {{ renderMember(type='time', name='GameTime') }}

:   Access the current in-game time
<!--tlo-forms-end-->

## Usage

```
/echo ${GameTime.Date}
```

Echos todays in-game date to the chat window.
