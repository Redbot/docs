# /zonehud

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/zonehud {on|off}
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Toggles loading the HUD section for the current zone. You must have a `[zone]` section in `MQ2HUD.ini`, long names are used.
<!--cmd-desc-end-->
