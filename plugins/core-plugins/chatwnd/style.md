# /style

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/style [!]0xN
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Set the style bit corresponding to 0xN or unset it if ! is supplied. See EQ documentation for further details about WindowStyle.
<!--cmd-desc-end-->
