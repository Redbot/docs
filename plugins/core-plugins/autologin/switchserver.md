# /switchserver

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/switchserver <server short name> <character name>
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Logs out of the current character and logs back into the specified server and character using the same account credentials.
<!--cmd-desc-end-->
