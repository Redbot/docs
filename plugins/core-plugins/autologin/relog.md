# /relog

## Syntax
<!--cmd-syntax-start-->
```eqcommand
/relog [<#>s|<#>m]
```
<!--cmd-syntax-end-->

## Description
<!--cmd-desc-start-->
Logs out the current character and automatically logs back in after 10 seconds or specified delay. e.g. `/relog 5m` for 5 minutes.
<!--cmd-desc-end-->
